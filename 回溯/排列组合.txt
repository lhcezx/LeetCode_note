组合问题中start是为了避免在组合的不同位置选择相同的元素; 排列问题中visited是为了在不同的位置选择相同的元素; 
如果组合或者排列问题的初始数组中有重复的数字，为了让组合和排列结果中不包含重复结果，应当将sort(nums), 然后增加判断条件。组合问题比较简单, if (i > start && nums[i] == nums[i - 1]) continue; 这个语句保证了在同一个位置x我们不会选择重复的数字来进行组合
排列问题会多一个条件，if (i > 0 && nums[i] == nums[i - 1] && !visited[i - 1]) continue;  在同一个位置x, 组合的条件会保证我们不选到同样的元素，但是当我们移动到下一个位置x + 1时，该条件会导致我们选不到两个重复元素的后一个元素, 也就是我们希望当visited[i - 1] == true
的时候，代表我们在进行正常的逻辑排列组合，保证我们可以让x+1的位置选到重复元素的后一个元素。同时也保证了当我们在x位置已经放置了重复元素的前一个元素nums[i - 1]时，for循环让我们来到nums[i], 由于for循环结束时我们会执行visited[i] = false， 因此此时visited[i - 1] = false，并且后一个元素和前一个相等，满足条件直接跳过
